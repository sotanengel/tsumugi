# Tsumugi (紡)

> Weave your story — OSS video editing for everyone.

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)
[![WIP](https://img.shields.io/badge/status-work_in_progress-yellow.svg)]()

**Tsumugi** is a free, open-source, privacy-first desktop video editor built
with [Tauri](https://tauri.app/), Rust, and React. From TikTok shorts to
wedding movies — edit locally, no cloud required.

## Status

This project is in **Phase 0** (foundation). Not yet usable for editing.
